# Prototype ATM
This app provides limited functions of ATM

## Main Features
    1. insert card
    2. select account
    3. withdraw
    4. deposit
    5. exit


## Prerequisite
Python 3.7.*

### Quick Start 

    1. git clone git@github.com:Yeongje/atm.git
    2. cd atm project
    3. pip install -r requirements.txt
    4. python main.py
